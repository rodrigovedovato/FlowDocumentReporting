------------- Flow Document Reporting Engine -------------------

This library that makes it easy to create data-driven reports in 
WPF by using the widely known FlowDocument. For now, it supports
the creation of simple tables and data-binded text fields.

V 1.0 - Only engine (example project comming soon). Supports the
creation of simples tables or any other databased list (the behavior
is similar to the asp.net datarepeater control)
